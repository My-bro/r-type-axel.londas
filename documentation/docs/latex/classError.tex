\doxysubsection{Detailed Description}
Base class for custom exception handling. 

This class serves as a base for specific error types. It inherits from std\+::exception and requires derived classes to implement the \doxylink{classError_a9e09deb110ba18bda3b5c2db283fe36b}{what()} method.
